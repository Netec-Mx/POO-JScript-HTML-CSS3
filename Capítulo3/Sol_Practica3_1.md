## Actividad — Corrección de errores en una página HTML5

### Objetivo

Al finalizar esta actividad, serás capaz de:

* Identificar errores comunes en documentos HTML.
* Detectar problemas de estructura y sintaxis.
* Corregir etiquetas mal escritas o incompletas.
* Validar formularios y enlaces básicos.
* Mejorar la correcta organización de una página web.

<br/><br/>

## Escenario

Un desarrollador junior creó la siguiente página HTML para un pequeño sitio web de servicios digitales.

Sin embargo, la página contiene **5 errores típicos de HTML**.

Tu tarea será:

1. Analizar el código.
2. Identificar los errores.
3. Corregirlos.
4. Ejecutar nuevamente la página en el navegador.
5. Verificar que el HTML funcione correctamente.

<br/><br/>

## Requisitos

| Herramienta        | Descripción                             |
| ------------------ | --------------------------------------- |
| Navegador web      | Chrome, Firefox o Edge                  |
| Visual Studio Code | Editor de código                        |
| Live Preview       | Extensión opcional para visualizar HTML |

<br/><br/>

# Página HTML con errores

> IMPORTANTE:
> La página debe ejecutarse primero SIN corregir.
> Después identifica y corrige los errores.

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi Página Web<title>
</head>

<body>

    <header>
        <h1>Diseño y Desarrollo Web</h1>
        <p>Servicios digitales modernos.</p>
    </header>

    <nav>
        <a href="#">Inicio</a>
        <a href="#">Servicios</a>
        <a href="#">Contacto</a>
    </nav>
    </nav>


    <main>

        <section>
            <h2>Servicios</h2>

            <article>
                <h3>Diseño Web</h3>
                <p>🎨</p>
                <p>Creación de páginas modernas.</p>
            </article>

            <article>
                <h3>Aplicaciones móviles</h2>
                <p>📱</p>
                <p>Desarrollo para Android e iOS.</p>
            </article>

        </section>

        <section>
            <h2>Contacto</h2>

            <form>

                <div>
                    <label>Nombre:</label>
                    <input type="text" nombre="nombre">
                </div>

                <div>
                    <label>Correo:</label>
                    <input type="email">
                </div>

                <div>
                    <input type="submit" value="Enviar"
                </div>

            </form>

        </section>

    </main>

</body>
</html>
```

<br/><br/>

# Instrucciones

## Tarea 1. Ejecutar la página

### Paso 1.

Crea un archivo llamado:

```text
index.html
```

### Paso 2.

Copia el código HTML anterior.

### Paso 3.

Abre la página en el navegador usando Live Preview o directamente con el navegador.

<br/><br/>

# Tarea 2. Detectar errores

Analiza cuidadosamente:

* Etiquetas HTML abiertas y cerradas.
* Atributos incorrectos.
* Sintaxis de formularios.
* Estructura de encabezados.
* Etiquetas incompletas.

<br/><br/>

# Tarea 3. Corregir la página

Corrige los 5 errores encontrados.

<br/><br/>

# Resultado esperado

La página deberá:

* Abrir correctamente.
* Mostrar todos los textos.
* No tener errores visuales importantes.
* Mostrar correctamente el formulario.
* Mantener una estructura HTML válida.
